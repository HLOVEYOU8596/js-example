### 服务启动工具

#### 作用：
在电脑上直接打开html文件，是以file:///方式打开的，这种方式很多时候会遇到跨域或者各种各样的问题，因此我们一般会搭建一个简易的本地服务器，来运行测试页面，这样子就可以正常预览观看放在本地的html网页或者动画了。

#### 功能：

给node server添加了外壳，原先是通过终端敲 node server.js来启动服务器，
现在只要双击start.bat就可以启动服务器了，用起来简易很多。

#### 使用方法：

1.双击打开 start.bat就是开启了一个服务器，关掉，就是把服务给关闭了；

2.把html页面放到public这个目录下，html名字改成index，然后用网页打开 http://127.0.0.1:1336/ ，就能正常预览了；
